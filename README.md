# EUROGP-2026
Artifacts for EuroGP 2026 conference for paper:
Multi-Action Tangled Program Graph for Multi-Task Reinforcement Learning with Continuous Control. Vacher Quentin, Beuve Nicolas, Dardaillon Mickaël, Desnos Karol.

paper is made available in this github repository before publication on HAL.

Results can be found in the _results.ipynb file.

 - [Environment](https://github.com/gegelati/gegelati-apps) repository with commit value: 6776a6405eae4ab2600886c0c1a4b786b38752f0
